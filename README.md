# SolarSystem
Solar System in Unity for the Modeling and Visualization subject

## Controls:
* G - general camera
* C - rotate cameras (multiple cams)
* M - meteor (multiple cams)
* N - spaceship (multiple cams)
* O - orbits
* P - planets (multiple cams)
